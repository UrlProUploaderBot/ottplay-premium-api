# OTTPlay Premium API (Vercel)

## Endpoint
/api/search

## Examples
/api/search?q=Salaar
/api/search?q=Bahuballi
/api/search?q=RRR&year=2022
/api/search?q=Jawan&page=2

## Features
- Auto latest year detect
- Wrong spelling support
- Movie + Web Series
- OTT platforms
- OTT release date
- Poster & description
